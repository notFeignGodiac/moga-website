# moga-website

A fully interactive multi-page website built with pure HTML, CSS & JavaScript. No frameworks, no libraries, no dependencies — just clean front-end code.

TO preview: DOWNLOAD THE MOGA-WEBSITE.HTML AND OPEN IT IN YOUR DESIRED BROWSER. 

## Pages
- Home
- Services
- Portfolio 
- Compliance & Legitimacy
- About
- Contact

## Features
- Custom color-reactive cursor that changes based on what's being hovered
- Scroll-triggered reveal animations
- Portfolio filter tabs
- Accordion FAQ
- Contact form with success state
- Mobile responsive with full-screen hamburger menu
- Scrolling marquee ticker
- Consistent 4-color brand design system

## Tech
- HTML5
- CSS3 
- Vanilla JavaScript
